Penjelasan & Implementasi

Apa itu jQuery?

jQuery adalah pustaka JavaScript yang menyederhanakan perintah rumit. Contohnya, tombol biru di bawah menggunakan fungsi fadeToggle() untuk menyembunyikan/menampilkan kotak penjelasan ini dengan halus.

Apa itu AJAX?

AJAX adalah teknik untuk memproses data di latar belakang. Halaman tidak perlu muat ulang (*refresh*) saat kita mengirim atau mengambil data baru. Ini membuat web terasa seperti aplikasi desktop.
