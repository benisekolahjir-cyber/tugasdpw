Kesimpulan:
Berdasarkan skenario uji coba, atribut name sangat penting pada input radio. Jika nilai name sama, browser akan mengelompokkannya sehingga kita hanya bisa memilih satu opsi (fungsi asli radio button). Jika nilai name berbeda, setiap pilihan dianggap berdiri sendiri dan kita bisa mencentang semuanya sekaligus.

Uji coba 3



Kesimpulan Uji Coba 3:
Input tipe checkbox memungkinkan pengguna untuk memilih lebih dari satu opsi (pilihan ganda) dalam satu waktu. Hal ini berbeda dengan radio button yang hanya mengizinkan satu pilihan saja. Setiap kotak centang bekerja secara mandiri sehingga kita bisa mencentang semua kotak atau tidak sama sekali.


Uji coba 4




Kesimpulan Uji Coba 4:
HTML menyediakan tipe input khusus seperti type="color" untuk memilih warna dan type="date" untuk kalender. Penggunaan type="button" dengan atribut onclick berfungsi untuk menjalankan perintah JavaScript, seperti memunculkan pesan peringatan (alert) saat tombol diklik.


Uji coba 5




Kesimpulan Uji Coba 5:
HTML5 menyediakan input yang lebih spesifik seperti datetime-local untuk memilih tanggal sekaligus jam, month untuk memilih bulan dan tahun saja, serta email yang secara otomatis memvalidasi format alamat email pengguna.


Uji coba 6



Kesimpulan Uji Coba 6:
Tipe image berfungsi sebagai tombol submit berupa gambar, file memungkinkan pengguna mengunggah berkas dari perangkat mereka, dan hidden digunakan untuk menyimpan data teknis (seperti ID pengguna) yang tidak perlu dilihat oleh pengguna di halaman web.
