Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan Ujicoba:
* **Tombol Reset**: Berfungsi untuk mengosongkan atau mengembalikan isi input ke nilai awal (default) yang ada pada atribut value. * **Tombol Kirim**: Digunakan untuk mengirim data ke server melalui file yang ditentukan di atribut action. * **Atribut Target**: Penggunaan _blank akan membuka hasil kiriman di tab baru, sedangkan _self di tab yang sama. * **Method GET vs POST**: method="get" mengirim data melalui URL sehingga terlihat oleh pengguna, sedangkan method="post" mengirim data secara tertutup (tidak terlihat di URL).
